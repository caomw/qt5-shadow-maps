qt5-shadow-maps
===

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

> Shadow mapping implementation with Qt5 and OpenGL (Qt 5.4 or higher)

## Build

Please use ```CMake 3.0.0``` or higher.

```shell
$ cmake .
$ cmake --build .
```
